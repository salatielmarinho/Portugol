# Portugol

## Começando

O Portugol Studio é um ambiente de programação didático voltado para iniciantes em programação que falam o idioma português. Ele oferece uma sintaxe fácil, diversos exemplos e materiais de apoio à aprendizagem. Além disso, possibilita a criação de jogos e outras aplicações. Aqui estão os principais recursos do Portugol Studio:

 - Interface Simples e Didática: O Portugol Studio possui uma interface fácil de usar, ideal para quem está começando a programar.
  
 - Sistema de Ajuda e Material de Apoio: Oferece suporte ao aprendizado com material de apoio, incluindo exemplos e documentação.
  
 - Atualização Automática: A ferramenta é atualizada automaticamente, garantindo acesso às últimas melhorias e correções.
Editor de Código Fonte Avançado: O editor possui recursos avançados para escrever e depurar programas.
Suporte a Bibliotecas e Desenvolvimento de Jogos: O Portugol Studio permite a criação de jogos e suporta bibliotecas.

---

### Pré-requisitos

* [Portugol Studio - Versão Desktop](http://univali-lite.github.io/Portugol-Studio/) - Utilizado para rodar códigos em Portugol direto na IDE instalada no computador
* [Portugol WebStudio - Versão Web](https://portugol.dev) - Utilizado para rodar códigos em Portugol direto no navegador
